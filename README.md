An advanced multimodal healthcare chatbot that utilizes vision and voice for interaction. The project is built using a four-phase technical architecture, integrating state-of-the-art tools like Meta’s Llama 3 Vision for medical analysis and OpenAI’s Whisper for speech-to-text transcription. Users can upload images and ask questions verbally, receiving spoken medical insights generated via ElevenLabs or Google Text-to-Speech.
